# NeuroSky-MindWave-Mobile-Connector
NeuroSky MindWave Mobile Connector is a software to connect and receive data from the Neurosky MindWave Mobile. Written in C# .NET

# Supported NeuroSky Mindwave Hardware
- MindWave Mobile
- MindWave Mobile 1.5
